Import Backup

Cảnh báo: Việc import backup sẽ thay thế toàn bộ dữ liệu hiện tại!

Database hiện tại sẽ được backup tự động trước khi restore.

Hướng dẫn sử dụng:
  • Xuất Backup: Nhấn nút "Tạo và Tải Backup" để tải file backup về máy
  • Import Backup: Chọn file backup (.zip) và nhấn "Import Backup" để khôi phục dữ liệu
  • File backup được tạo tự động với tên: backup_inventory_YYYYMMDD_HHMMSS.zip
  • Hệ thống sẽ tự động backup database hiện tại trước khi import
